### [⚙️ How to run application ](#orm)
  #### Step 1:
   - Download project by using following command
   ```
   git clone https://github.com/krishnawisely/facedetection-js.git
   ```
  #### Step 2:
   - Run application by using following command
   ```
   npm i
   npm run start
   ```
